# reveal.jsとは

### Markdown対応のプレゼンテーションを手軽に作成できるフレームワーク
* GitHub から reveal.js を clone してくる
* Visual Studio Code を使って拡張機能をインストール

--

## 特徴
* HTMLでスライドを作成してブラウザ上でプレゼンを実行できる
* Markdownで作成可能
* 4方向にスライド遷移
